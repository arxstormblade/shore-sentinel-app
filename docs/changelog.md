# Shore Sentinel Product Roadmap Release Notes

## v0.3.0 - Product-roadmap epic release

### User-facing changes
- Guided scan entry flow from dashboard and Scans & Reports.
- Next-best-action guidance across dashboard, scan/report, and remediation surfaces.
- Remediation workflow states, teamwork detail, comments, evidence, and activity history.
- Report-to-remediation continuity and actionable saved operational views.
- Inline severity/score explanations and clearer prioritization guidance.
- Display preferences for density, contrast, and reduced effects.
- Trend analysis and internal posture benchmarking.
- Notification and scheduled-summary planning documented for future operations.

### Validation expectations
- Keep Kanban as the source of truth for epic/product work.
- Use Archon Protocol for concise owner progress/blocker updates.
- Run API/web/worker checks, Docker smoke, Tailnet live route checks, and Athena QA before calling the release complete.
